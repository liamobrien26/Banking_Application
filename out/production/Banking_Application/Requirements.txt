Objective
Create an application that allows users to make bank transactions (deposits and withdrawals) and print a bank statement to the terminal.

Requirements:
Implement Deposits and Withdrawals

Users should be able to deposit money into their account.
Users should be able to withdraw money from their account.
Ensure the account balance is updated correctly after each transaction.
Account Statement

Print a statement that includes:
A heading row.
Date of each transaction.
Amount of each transaction.
Balance after each transaction.
Transactions should be displayed in reverse chronological order (most recent first).
